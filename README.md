# WMMT Toolbox

为更方便的游玩Wangan Midnight Maximum Tune而生

### 功能

- 游戏启动器（包括启动Maxi Terminal, AMAuthd.exe, TeknoParrotUi）

- 网络修复（可修复hosts网络配置与TeknoParrot的网络配置）

- 游戏常见问题修复（如：2401报错、白屏等问题）

- 实用工具集（如：分辨率修改、刷卡键修改等功能）

### 关于

- 任何人都可以使用本工具箱，并且也可以修改源码二次生成

- 工具箱的关于页面有友情链接，若有WMMT服务器管理者希望将自己的群也放入进去的话，可以联系我或提交PR进行加入
